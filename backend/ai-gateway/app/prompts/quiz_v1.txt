## Format de sortie
Renvoie un JSON valide avec la structure exacte suivante, sans texte supplémentaire :

{{
  "quiz": {{
    "title": "Titre du quiz",
    "subject": "{subject}",
    "grade_level": "{grade_level}",
    "difficulty": "{difficulty}",
    "questions": [
      {{
        "id": 1,
        "type": "qcm",
        "question": "Texte de la question",
        "options": ["A) ...", "B) ...", "C) ...", "D) ..."],
        "correct_answer": "A",
        "explanation": "Explication pédagogique de la bonne réponse"
      }},
      {{
        "id": 2,
        "type": "vrai_faux",
        "question": "Affirmation à évaluer",
        "options": ["Vrai", "Faux"],
        "correct_answer": "Vrai",
        "explanation": "..."
      }}
    ]
  }}
}}

Pour "reponse_courte", le champ "options" est un tableau vide [] et "correct_answer" est la réponse attendue.

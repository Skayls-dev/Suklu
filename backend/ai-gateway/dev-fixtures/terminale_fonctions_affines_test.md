# Fonctions affines - fiche de rappel Terminale

Une fonction affine est une fonction definie par la formule `f(x) = ax + b`.

- `a` est le coefficient directeur.
- `b` est l'ordonnee a l'origine.
- Si `a > 0`, la droite est croissante.
- Si `a < 0`, la droite est decroissante.
- Si `a = 0`, la fonction est constante.

Pour reconnaitre une fonction affine, on cherche une expression de la forme `ax + b`.

Exemple 1 :

`f(x) = 3x + 2`

- coefficient directeur : `3`
- ordonnee a l'origine : `2`

Exemple 2 : cout d'une course de taxi.

Dans une ville, une compagnie de taxi facture un forfait de depart de `500 FCFA` puis `200 FCFA` par kilometre.
Si `x` est le nombre de kilometres parcourus, alors le prix `P(x)` est :

`P(x) = 200x + 500`

Cette expression est une fonction affine.

Interpretation :

- `200` represente l'augmentation du prix pour chaque kilometre supplementaire.
- `500` represente le prix a payer meme si la distance est nulle.

Representation graphique :

- la courbe representative d'une fonction affine est une droite ;
- elle coupe l'axe des ordonnees au point `b`.

Vocabulaire cle pour Terminale : fonction affine, coefficient directeur, ordonnee a l'origine, droite, variation, expression `ax + b`.
